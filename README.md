# ZK Shielded Pool

It is and zk shielded pool implementation in rust using halo2 for academic purposes.

# Directory Structure

```
zk-shielded-pool/
├── src/
   ├── circuit/
          └── constraint_1/
          └── constraint_2/
          └── constraint_3/
          └── constraint_4/
          └── constraint_5/
          └── constraint_6/
          └── constraint_7/
└── README.md
```

Main Circuit directory contains final circuit implementation and full circuit tests.
Each subdirectory contains a single constraint implementation with its own tests.
